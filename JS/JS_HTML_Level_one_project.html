Js Simple Basic Project

For this project we need to perform basic task.

  1. Accept First and Last name. First letter of first name and last name should be equal.
  2. Accept age and check whether its between 20 and 30 years.
  3. Accept height and height should be atleast 170cm.
  4. Accept Pet name and pet name should end with letter "y"

If all the condition are matched then print message as welcome spy in the console.
